## ボタン押下時の BOT 応答

### 「記録する」タップ

```
📝 記録モードです。
記録したい内容を送ってください（食事写真・テキスト・体重数値・体重計の写真、どれでもOK）
```

受理できるもの:
- 体重を数値で送る（例: `65.5kg`）
- 体重計の写真を送る
- 食事写真を送る
- テキストで食事を書く（例: `昼はそばでした`）

### 「相談する」タップ

```
💬 相談モードです。
食事・体重・外食・間食・続け方など、何でも送ってください 😊
```

### 「ダッシュボード」タップ

LIFF が直接開く（BOT 応答なし）。遷移フロー:

```
Rich Menu タップ
  → https://liff.line.me/2009409790-DekZRh4t
  → /liff (LIFF SDK 初期化 → LINE ログイン → JWT 発行)
  → /dashboard (ユーザーダッシュボード表示)
```

## 運用ルール（SSOT）

- **リッチメニューは入口だけ** — 3ボタンに限定
- **記録の中で食事と体重を分けすぎない** — 1つの入口に集約
- **問診やり直しはテキストコマンド** — `問診やり直し` で対応（低頻度のためメニュー不要）
- **ダッシュボードは必ず LIFF 直遷移** — 会話を挟まない
- **状態管理（招待コード・問診・画像確認）は決定論で守る**
